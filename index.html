The Big Deal 3D

Role: Web Developer

What started off as passion project in 2017 amoung a group of friends is now an annual music game event in Dallas, Texas. The Big Deal series is the largest rhythm game focused event in the United States with competitors traveling from all over the country and internationally to compete in various rhythm games.

My involvement with The Big Deal was recreating their website from scratch using modern practices in order to create a more user friendly experience and running one of the events featured.

The Big Deal: 3D was a huge success with over 300 competitors and over 100 spectators attending from March 29th - March 31st of 2019 with the next event planned for Q2 2020.

PumpOut

Role: UI/UX Engineer

Extensive tool for popular arcade game "Pump it Up" for players to gather more information about the game's songs and stepcharts in all in one place. PumpOut also featured a song randomizing feature that was widely used in different tournaments across the United States since 2016.

My involvement with the new version of PumpOut was creating user stories for a "tournament mode" feature and updating the user interface to be mobile responsive and more user friendly.
