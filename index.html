The Big Deal 3D

What started off as passion project amoung a group of friends is now an annual music game event in Dallas, Texas. The Big Deal series is one of the largest multi-rhythm game events in the United States with over 200 competitors traveling from all over the country and internationally to attend to compete in various games.

The next event will be hosted on March 29th - March 31st, 2019

Rock Paper Scissors Lizard Spock (Multiplayer)

Inspired by the TV show "The Big Bang Theory," this is a web browser version of the "Rock Paper Scissors" game featured in the series. This application works in real-time and is played by having two players accessing the link at the same time (or in two seperate windows if you are on the same machine) with a real time chat feature as well.

Languages used: HTML CSS, Javascript, jQuery and Google Firebase

View application

How I Met Your Mother Trivia Game

Web browser trivia game with a theme based on the TV show "How I Met Your Mother." Players have the opportunity to play 10 random questions or all 20 questions featured on the application. Players receive feedback in the form of random gifs from the show and will need to answer their questions within a designated time limit or else...

Languages used: HTML, CSS, Javascript and jQuery

View application

Eat-Da-Burger

Full stack web application where users can create a burger, type in their name and devour it. This information is stored serverside and will remain on the page in future visits.

Technologies used: HTML, CSS, Twitter Bootstrap, Javascript, Heroku, Node.js, MySQL Dependencies: Express.js, express-handlebats, path, body-parser and sequelize

Codiacs Travel Blog

This travel blog application was the result of a group effort by myself and 3 other developers. This travel blog application will help the travellers to look for blogs written about a specific city and see what type of blog categories exist for the city searched. User of the application can read,write and share their own blogs about their travelling experiences

Languages used: HTML, CSS, Javascript and jQuery
